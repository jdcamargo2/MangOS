# 01_pem

`01_pem` contiene el módulo táctico de ejecución de MangOS.

PEM significa **Protocolo de Extracción Mercenaria Académica**. Su función es convertir tareas, obligaciones, proyectos y pendientes en contratos claros, realistas y ejecutables.

Dentro de MangOS, PEM no es el sistema completo: es el módulo encargado de la acción.

## Función

PEM responde a preguntas como:

- ¿Qué contrato toca ahora?
- ¿Qué vence primero?
- ¿Qué es crítico, rentable o decorativo?
- ¿Qué debo tratar como Forja, Extracción o Trámite?
- ¿Cuál es el próximo paso mínimo?
- ¿Qué se liquida, qué se arrastra y qué se archiva?

## Archivo principal

- `pem_manual.md`: manual operativo completo del sistema PEM.

## Carpetas

- `contratos/`: contratos activos, diarios, semanales o especiales.

## Regla principal

PEM no debe convertirse en una carga burocrática.

Su objetivo es reducir fricción, no aumentarla.

## Flujo básico

1. Detectar tareas.
2. Convertirlas en contratos.
3. Clasificarlas por tipo y prioridad.
4. Elegir modo de energía.
5. Ejecutar el próximo paso mínimo.
6. Registrar bitácora.
7. Cerrar, arrastrar o archivar.
