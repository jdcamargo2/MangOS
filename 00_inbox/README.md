# 00_inbox

`00_inbox` es la bandeja de entrada rápida de MangOS.

Aquí se guarda todo lo que todavía no tiene una ubicación clara, pero que no debe perderse: notas sueltas, ideas incompletas, preguntas mal formadas, enlaces, capturas, materiales pendientes de revisar o fragmentos útiles de conversaciones.

## Función

Capturar primero, clasificar después.

El objetivo de esta carpeta no es ordenar perfectamente, sino evitar que información útil quede en el aire.

## Qué puede entrar aquí

- Notas rápidas.
- Ideas sin desarrollar.
- Preguntas incompletas.
- Links pendientes.
- Capturas transcritas.
- Materiales por revisar.
- Fragmentos de conversaciones útiles.
- Posibles proyectos todavía no definidos.
- Investigaciones que aún no tienen carpeta propia.

## Qué no debe quedarse aquí

- Proyectos ya definidos.
- Investigaciones ya clasificadas.
- Contratos PEM activos.
- Bitácoras formales.
- Decisiones importantes ya cerradas.
- Archivos que ya tienen una carpeta clara dentro de MangOS.

## Regla principal

Si todavía no sé dónde va, entra aquí.

Si ya sé dónde va, debe moverse a su carpeta correspondiente.

## Flujo de clasificación

- Si es una tarea ejecutable, mover a `01_pem/`.
- Si es un proyecto, mover a `02_proyectos/`.
- Si es una investigación, mover a `03_investigaciones/`.
- Si es una pregunta, mover a `04_preguntas/`.
- Si es una idea, mover a `05_ideas/`.
- Si ya no está activo pero conviene conservarlo, mover a `99_archivo/`.

## Revisión

El inbox debe revisarse periódicamente para evitar que se convierta en un basurero permanente.
