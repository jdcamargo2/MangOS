# Mapa de Repositorios de MangOS

MangOS es el sistema central de organización, narrativa y archivo de la República de Mangolandia.

Este documento define qué proyectos existen como repositorios independientes, cuáles viven como carpetas internas y cómo se relacionan entre sí.

## Repositorios y módulos

| Proyecto / Sistema | Decisión | Relación con MangOS | Razón |
|---|---|---|---|
| MangOS | Repo independiente | Sistema central | Centro de organización, narrativa, archivo y mapa general |
| PEM | Carpeta dentro de MangOS | Módulo táctico | Sistema de contratos, enfoque y ejecución |
| workflow-system | Repo independiente | Proyecto conectado | Sistema técnico con código, automatización y vida propia |
| Elíejesresce K2 | Repo independiente | Proyecto mayor referenciado por MangOS | Investigación cuántica principal y proyecto de vida técnica |
| distributed_dqc_prototype | Repo independiente | Satélite de K2 | Subproyecto cuántico completo nacido del mundo K2 |
| MTS | Repo independiente futuro | Proyecto registrado en MangOS | Proyecto de bajo nivel sobre C, hilos y sistemas |
| VulneraData-Mérida | Repo independiente | Proyecto registrado | Proyecto de datos, territorio y vulnerabilidad |
| Sistema-gestion-viajes | Repo independiente | Proyecto académico registrado | Proyecto universitario con nota y valor curricular |
| Proyecto-Web-PuntoPymes | Repo independiente colaborativo | Proyecto registrado | Proyecto web grande desarrollado en colaboración |
| Gestion-agua-genero | Repo independiente | Investigación / documentación | Base de investigación del proyecto agua y género |
| gestion_agua_genero | Repo independiente | Web / despliegue | Página web asociada al proyecto agua y género |
