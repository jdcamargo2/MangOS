# MangOS Command Center

MangOS Command Center es una aplicación nativa de Windows pensada como centro de mando personal para capturar ideas, preguntas, contratos, aprendizajes y tareas, organizarlas automáticamente y acompañar sesiones de trabajo mediante el sistema PEM.

## Objetivo

Construir una herramienta minimalista, visual y eficiente que permita:

- Capturar información en lenguaje natural.
- Clasificar ideas, preguntas, contratos y aprendizajes.
- Gestionar contratos de trabajo.
- Cerrar sesiones PEM automáticamente.
- Generar bitácoras, arrastres y continuidad.
- Integrarse progresivamente con APIs externas.

## Stack tentativo

- C# + WinUI 3
- Windows App SDK
- SQLite
- APIs web
- Integración futura con IA, TickTick, calendario y Workflow System
