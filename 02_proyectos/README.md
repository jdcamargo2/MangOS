# 02_proyectos

Carpeta para proyectos vivos de MangOS.

Aquí van sistemas, prototipos, laboratorios, investigaciones y líneas de trabajo que tienen entidad propia.  
No todo lo que está aquí está activo al mismo tiempo: algunos proyectos están en desarrollo, otros en pausa y otros quedan como archivo técnico para retomar después.

## Regla de uso

Un proyecto entra aquí cuando:

- tiene carpeta propia;
- tiene continuidad posible;
- puede generar documentos, código, bitácora, imágenes o prototipos;
- no es solo una idea suelta de la bandeja.

Si una idea todavía no tiene forma clara, primero va a `00_inbox`.

## Estados

- **Activo:** proyecto con movimiento reciente o prioridad actual.
- **Standby:** proyecto válido, pero no está en caza ahora.
- **Archivado:** proyecto conservado como referencia, sin intención inmediata de retomarlo.
- **Exploratorio:** proyecto todavía en definición.
